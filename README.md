# 💳 Simple Banking System

A simple console-based banking application built with Java to demonstrate core Object-Oriented Programming (OOP) concepts.

## 🚀 Features

- Create an account with user input
- Deposit money
- Withdraw money with balance validation
- View account details
- Interactive menu system

## 🛠️ Technologies Used

- Java
- OOP (Object-Oriented Programming)
- Scanner (user input handling)

## 📚 What I Learned

- Creating classes and objects in Java
- Using constructors
- Writing methods (deposit, withdraw, display)
- Applying conditional logic (if/else)
- Building interactive console applications
- Basic Git & GitHub workflow

## ▶️ How to Run

```bash
cd src
javac Main.java Account.java
java Main
```
